# Review Notes

Найденные замечания перед рефакторингом:

1. `GameManager` содержал слишком много ответственности: подготовку сессии, обработку ввода, цикл игры и отрисовку.
2. В `PlayerPreparationService` использовались магические числа и строковые литералы для стартовых параметров игрока и оружия.
3. README не содержал описания архитектуры и списка примененных паттернов.
4. Длинные методы усложняли чтение и повышали риск регрессий при изменениях.

Что исправлено:

1. В `GameManager` применен `Extract Method`: выделены методы подготовки цикла, очистки консоли, спавна врага, отрисовки HUD и арены.
2. Магические числа и строки заменены именованными константами в `PlayerPreparationService` и `GameManager`.
3. README обновлен разделами `Архитектура` и `Используемые паттерны`.
